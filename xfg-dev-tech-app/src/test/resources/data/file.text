2023年11月15日凌晨2:17，伦敦大英博物馆的安防系统突然触发。当保安队长詹姆斯·威尔逊赶到亚洲文物展厅时，防弹玻璃展柜已被切割，镇馆之宝——明朝永乐年间的翡翠龙纹王冠不翼而飞。监控显示，盗贼全程戴着威尼斯狂欢节面具，手法专业，仅用3分42秒便完成盗窃。

​第二章：神秘收藏家的晚宴​
盗窃案发生48小时后，苏富比拍卖行前鉴定专家艾琳·周收到匿名邀请函，要求她参加瑞士苏黎世湖别墅的私人鉴赏会。晚宴主人是俄罗斯寡头维克托·伊万诺夫，他展示了三件与失窃王冠风格相似的翡翠饰品，并暗示自己掌握着"永乐宝藏"的线索。

​第三章：仰光的暗网交易​
缅甸仰光，黑客"暹罗猫"在暗网论坛发现有人兜售文物运输路线加密数据包，标价50比特币。交易记录显示买家IP曾多次登录香港中环某古董店的WiFi。当地线人报告，最近一周有说俄语的外国人频繁出入密支那翡翠矿区。
